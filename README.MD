m.combee.co
========

You will need:
* ```node```  --> [http://nodejs.org]()
* ```npm``` --> [https://npmjs.org]()
* ```yeoman``` --> ```npm install yo -g```

Run:

1. ```npm install```
2. ```bower install```
3. ```grunt serve```

Find your computer's IP address, point your mobile brower to it's ```9000``` port.

There you go.
